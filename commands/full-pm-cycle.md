# Full PM Cycle Workflow

Complete product management from idea to impact across all five layers.

## What This Does

Orchestrates the complete product lifecycle:
- **Layer 1 (Perception)**: Market research, user research, competitive analysis
- **Layer 2 (Strategy)**: Positioning, roadmap, prioritization
- **Layer 3 (Design)**: PRD generation, prototype design
- **Layer 4 (Delivery)**: Requirement review, project coordination, release
- **Layer 5 (Validation)**: Impact analysis, feedback synthesis

## Usage

- `/full-pm-cycle "new project management tool"`
- `/ompm full-pm-cycle "AI assistant feature" new-feature`

## Execution

This command activates the `full-pm-cycle` workflow skill.

Your input: $ARGUMENTS
