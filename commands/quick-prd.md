# Quick PRD Workflow

Generate a complete PRD with competitive analysis in one workflow.

## What This Does

Orchestrates scenario detection, competitive analysis (optional), and PRD generation:
1. **Scenario Detection**: iteration/new-feature/new-product
2. **Competitive Analysis**: optional feature comparison
3. **PRD Generation**: with industry benchmarks

## Usage

- `/quick-prd "user profile redesign" Taobao JD`
- `/ompm quick-prd "dark mode for mobile app"`

## Execution

This command activates the `quick-prd` workflow skill.

Your input: $ARGUMENTS
