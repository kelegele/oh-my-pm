## 重复出现的流程模式

### 常见瓶颈
- **审批流程**: 单一审批人造成等待
- **信息不对称**: 需求不清晰导致返工
- **工具切换**: 多个工具间切换浪费时间

### 优化模式
- **并行化**: 独立任务同时执行
- **自动化**: 手动工作用脚本/工具替代
- **简化**: 移除不增值的步骤

## 浪费类型框架

### DOWNTIME
- **D**efects 缺陷 - Bug、错误
- **O**verproduction 过度生产 - 未使用的功能
- **W**aiting 等待 - 审批队列
- **N**on-utilized talent 人才未利用
- **T**ransportation 搬运 - 文件传输
- **I**nventory 库存 - 大量积压
- **M**otion 动作 - 工具切换
- **E**xtra processing 过度处理 - 冗余审批

## 优化技术

### 价值流映射
1. 绘制当前状态流程
2. 识别增值/非增值活动
3. 设计未来状态
4. 实施改进

### 瓶颈分析
1. 识别约束点
2. 量化等待时间
3. 分析根因
4. 消除或缓解
