## 重复出现的用户模式

### 按用户类型
- **产品经理**: 关注效率、团队协作、优先级
- **开发者**: 关注自动化、减少会议、文档清晰
- **设计师**: 关注设计一致性、组件库
- **创业者**: 关注成本效益、快速上线

### 常见痛点
- 工具碎片化，需要切换多个应用
- 信息不对称，沟通成本高
- 流程复杂，学习成本高

## 有效访谈技巧

### 开场问题
1. 请介绍一下你的角色
2. 典型的一天是怎样的？

### 深度挖掘
3. 你目前如何解决 [问题 X]？
4. 什么让你感到困扰？
5. 理想情况会是什么样的？

### 结束
6. 如果只能改变一件事，会是什么？
